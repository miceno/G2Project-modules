Coppermine->Gallery2 migration module.

Installation:
* Unpack the archive
* Copy the resulting cmmigrate directory to the modules directory of your Gallery2
  installation
* As site admin, install and activate the module in the Site Admin/Plugins menu

Useage:
* As site admin go to the Import/Coppermine menu choice
* Enter the path to your main Coppermine directory, the module will 
  read the settings from the config file automatically
* Work your way through the provided screens
